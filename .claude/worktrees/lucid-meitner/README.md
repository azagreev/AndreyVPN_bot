# AndreyVPN_bot 🛡️

Telegram-бот для удобного управления VPN-сервером **AmneziaWG** на базе Ubuntu 22.04. Проект предназначен для личного использования и предоставления доступа друзьям с полным контролем со стороны администратора.

## Основные возможности

### 🔒 Безопасность и доступ
- **Защита от ботов**: Автоматическая CAPTCHA при первом запуске.
- **Белый список (Whitelist)**: Доступ к функциям бота только после ручного одобрения администратором.
- **Двухэтапное подтверждение**: Каждая выдача VPN-конфига требует отдельного аппрува админа.

### 🔑 Управление VPN (AmneziaWG)
- **Генерация профилей**: Автоматическое создание пар ключей и выбор свободного IP-адреса.
- **Удобная доставка**: Выдача готового профиля в виде **QR-кода** (для мобильных устройств) и файла **.conf** (для ПК).
- **Поддержка обфускации**: Конфиги генерируются с учетом параметров AmneziaWG (Jc, Jmin, Jmax, S1, S2, H1-H4) для обхода блокировок.

### 📊 Мониторинг и статус
- **Статистика трафика**: Пользователи могут видеть свое потребление за текущий месяц через команду `/stats`.
- **Ежемесячный сброс**: Автоматическое обнуление счетчиков трафика 1-го числа каждого месяца.
- **Статус сервера**: Администратор может проверить состояние WireGuard-интерфейса и количество активных пиров командой `/server`.

## Технологический стек

- **Язык**: Python 3.12
- **Библиотека бота**: [aiogram 3.x](https://github.com/aiogram/aiogram)
- **База данных**: SQLite (aiosqlite)
- **Логирование**: Loguru
- **Генерация QR**: Segno
- **Конфигурация**: Pydantic Settings

## Быстрый старт

### 1. Требования
- ОС: Ubuntu 22.04 (или совместимая с WireGuard/AmneziaWG).
- Установленная утилита `wg` (wireguard-tools).
- Настроенный интерфейс AmneziaWG (например, `wg0`).

### 2. Установка
```bash
git clone https://github.com/azagreev/AndreyVPN_bot.git
cd AndreyVPN_bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Настройка
Скопируйте пример конфига и заполните его:
```bash
cp .env.example .env
nano .env
```
**Важные параметры:**
- `BOT_TOKEN`: Токен от @BotFather.
- `ADMIN_ID`: Ваш Telegram ID.
- `SERVER_PUBLIC_KEY`: Публичный ключ сервера.
- `SERVER_ENDPOINT`: IP:PORT вашего сервера.
- `WG_INTERFACE`: Имя интерфейса (обычно `wg0`).

### 4. Запуск
```bash
python3 main.py
```

## Список команд
- `/start` — Регистрация и прохождение капчи.
- `/menu` — Главное меню управления (запрос VPN).
- `/stats` — Просмотр потребления трафика за месяц.
- `/server` — (Для админа) Статус сервера и количество пиров.
